# Binary Search Visualizer

An interactive React app for visualizing binary search step by step.

## Live Demo

[Open the app](https://deekshapaic.github.io/binary-search-visualizer/)

## Features

- Visualize iterative and recursive binary search
- Compare binary search with linear search
- Step through the search process
- Try best, worst, and average case examples
- Export the result as a PDF

## Run Locally

```sh
npm install
npm start
```

If your Node version is newer and the build fails with an OpenSSL error, run:

```sh
NODE_OPTIONS=--openssl-legacy-provider npm run build
```
