Binary Search Guide

How it works

Binary search works on a sorted array. It checks the middle value, then removes half of the remaining search space after every comparison.

Steps

  1. Start with low at the first index and high at the last index.
  2. Find the middle index.
  3. If the middle value is the target, the search is complete.
  4. If the target is larger, continue on the right half.
  5. If the target is smaller, continue on the left half.

Time complexity

The time complexity is O(log n) because the search space is divided by two each step.
